A empresa SegSoft atua como prestadora de serviços de software, sendo uma filial brasileira de uma organização matriz sediada em Berna na Suíça. Os clientes desta empresa estão distribuídos em muitos países distintos, sendo que cada qual está sujeito a diferentes formas de tributação quanto aos serviços prestados. A SegSoft possuí um sistema próprio para gestão dos dados de suas notas fiscais, sendo estes armazenados em uma base de dados estruturada, utilizando a tecnologia MySQL. Entretanto, devido ao grande volume de dados existente nesta plataforma, alinhado a necessidade de constantes consultas, a organização decidiu replicar estes dados em uma base de dados NoSQL a fim de proporcionar maior velocidade de acesso aos dados e robustez quanto a sua disponibilidade. Neste contexto, você, como analista de sistemas que conhece as tecnologias NoSQL, precisa escrever uma rotina computacional para migração dos dados da base relacional (MySQL) para o banco de dados NoSQL (Apache Cassandra). Uma vez realizada esta migração, você ainda precisa desenvolver uma funcionalidade cujo objetivo é imprimir em formato PDF as notas fiscais emitidas. Esta funcionalidade deve receber como parâmetro o número de uma nota fiscal, e então produzir uma versão imprimível que deve ser apresentada em um arquivo PDF. Este documento deve conter:

  • Nome Cliente
  • Endereço Cliente
  • Itens da nota
    • Descrição do serviço
    • Quantidade
    • Valor unitário
    • Nome do recurso que prestou o serviço
    • Função do recurso que prestou o serviço
    • Taxa/Impostos
    • Desconto
